Welcome to the BioJSON community site.

BioJSON is a JavaScript Object Notation (JSON) specification for the representation and exchange of bioinformatics data. The first version of the specification was developed in 2015 and implemented in Jalview 2.9.

Get involved

Our aim with BioJSON is to allow mobile, web-based and standalone bioinformatics applications to efficiently and easily exchange data and metadata. If you have any suggestions or want to contribute to the BioJSON standard, then get in contact or file an issue.

Why use BioJSON

  • JSON is very popular with established parsers in most programming languages
  • BioJSON v1 allows multiple sequence alignments as well as quantitative and symbolic sequence and alignment annotation to be represented. It also encodes rendering and display styles
  • BioJSON can be embedded as supporting data for interactive web pages
  • Integrates seamlessly with the HTTP rest architecture
  • Very light weight, self describing and easy to understand

We think it is a generally useful format to use for representing alignment and associated data on the web. It provides a way of handling single multiple-sequence alignments and annotations, which enables interoperation between stand-alone, web based and mobile multiple-sequence alignment and analysis systems.

What does BioJSON support?

BioJSON v1 describes single alignments with their annotations, features, groups, hidden sequences, hidden columns and colour schemes. It also allows custom application settings to be encoded for the Jalview multiple alignment workbench.

Future versions of BioJSON will provide more complete support for the interchange of multiple alignment data. It will also introduce new representations for phylogenetic data and 3D structures.

BioJSON v1 UML Diagram

Some UML notations and their meanings

  • 0..1 → zero or one
  • 0..* → zero or many
  • 1..1 → one to one
  • 1..* → one or many

The BioJSON Schema

BioJSON is specified using version 4 of the jsonschema standard. The schema is available via the github repository or can be downloaded from here. You can explore the BioJSON schema with the interactive visualisation created with Docson.

Instructions for interacting with the Docson schema

  • When the mouse is hovered over the elements in the BioJSON schema section, traversable elements become highlighted and clickable
  • Click an object’s “object” button to expand it, or the plus “+” link to expand all objects
  • Click the minus “-” to collapse all objects
  • The parenthesis “{}” link shows the underlying JSON schema which provides a formal description of the BioJSON format
  • Expanding an object reveals its variables, their type and a brief description in the object, their data types and the description of each variable

See BioJSON in Action!

The current application of BioJSON in Jalview 2.9 includes:
  • Export and import of BioJSON data as JSON flat files
  • Export and import of supporting data for interactive web pages, including:
    • Interactive SVG figures (File → Export Image → HTML)
    • The BioJS MSA viewer (File → Export Image → BioJS)

Future capabilities

Some of our goals for future versions of BioJSON are:
  • BioJSON services for Jalview desktop
  • BioJSON for the Javascript version of Jalview (JalviewJS)
  • Representation of linked CDNA and Protein alignments views
